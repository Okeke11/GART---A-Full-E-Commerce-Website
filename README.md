# Gart

A secure, multi-vendor e-commerce marketplace built with **Node.js, Express, and MongoDB**.

## Key Features
* **🛒 Marketplace:** Buyers can browse, cart, and purchase products from multiple sellers.
* **🔒 Escrow System:** Payments are held securely until the buyer confirms receipt via a unique **Delivery OTP**.
* **💰 Seller Dashboard:** Sellers manage products, verify deliveries, and receive auto-payouts to linked bank accounts.
* **🔔 Notifications:** Real-time alerts for orders, sales, and payouts.

## How to Run
1.  **Start MongoDB** (Ensure `mongod` is running).
2.  **Install Dependencies:**
    ```bash
    npm install
    ```
3.  **Start Server:**
    ```bash
    node server.js
    ```
4.  **Visit:** `http://localhost:3000`
